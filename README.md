# 德州扑克联机游戏

一款使用 Go 后端 + Tauri 桌面客户端实现的实时多人德州扑克游戏。

## 功能特性

- **房间系统**：创建房间、加入房间（6位房间码）、实时玩家列表
- **完整牌局**：Pre-flop → Flop → Turn → River → Showdown
- **五种动作**：过牌(Check)、跟注(Call)、加注(Raise)、弃牌(Fold)、All-in
- **十种牌型**：皇家同花顺 → 高牌，正确判断大小
- **结算系统**：主底池分配、赢家判定、净盈亏统计
- **操作历史**：左侧可伸缩面板，实时显示所有玩家动作

## 快速开始

### 1. 启动服务端

```bash
cd server
go build -o main ./cmd/server
./main
```

服务端口：
- WebSocket: `8080`
- HTTP: `8081`

### 2. 启动客户端

```bash
cd client
npm install
npm run tauri dev
```

### 3. 开始游戏

1. 玩家 A：输入名称 → 创建房间 → 获得房间码
2. 玩家 B：输入名称 → 加入房间 → 输入房间码
3. 玩家 A：点击开始游戏
4. 轮流进行动作选择

### 游戏动作

| 动作 | 说明 | 快捷键 |
|------|------|--------|
| 过牌 | 无人加注时跳过 | 空格 |
| 跟注 | 下与当前最高注相同 | 空格 |
| 加注 | 超过当前最高注 | A |
| 弃牌 | 退出本轮 | F |
| All-in | 押上全部筹码 | - |

### 牌型大小

| 等级 | 牌型 | 说明 |
|------|------|------|
| 9 | 皇家同花顺 | A♠K♠Q♠J♠10♠ |
| 8 | 同花顺 | 同花+顺子 |
| 7 | 四条 | 四张同点 |
| 6 | 葫芦 | 三条+对子 |
| 5 | 同花 | 同花色 |
| 4 | 顺子 | 连续点数 |
| 3 | 三条 | 三张同点 |
| 2 | 两对 | 两个对子 |
| 1 | 一对 | 一个对子 |
| 0 | 高牌 | 单张最大 |

## WebSocket 协议

消息格式：`{"type": "action_type", "data": {...}}`

**客户端 → 服务端**
- `create_room` - 创建房间
- `join_room` - 加入房间
- `leave_room` - 离开房间
- `start_game` - 开始游戏
- `player_action` - 玩家动作

**服务端 → 客户端**
- `room_update` - 房间状态更新
- `game_state` - 游戏状态推送
- `game_result` - 结算结果
- `error` - 错误信息
